주요 업무
- 쇼핑몰 사용자 사이트 신규 개발
- 홈, 상품리스트, 이벤트, 기획전 페이지 개발
- 5depth 계층형 카테고리 상품 필터 시스템 구현
- 로그인·회원가입·내정보 수정·SNS 로그인 등 인증 관련 기능 개발
- 프론트엔드 팀 리빌딩 및 신규 인력 온보딩 주도

기술 스택
- Core: React 17, Nextjs 12 (Page Router), Typescript
- Library: Tanstack Query, React Hook Form

핵심 성과
- -성능개선: 페이지 전체 블로킹을 제거하여 FCP 0.6초 달성
- 상품필터: 5depth 계층 구조를 지원하는 필터 시스템 구현
- 팀 리빌딩: 팀원 전원 퇴사 상황에서 인력 재구성 후 일정 내 프로젝트 완수


온라인 의류 쇼핑몰 사용자 사이트 신규 개발
[FCP 0.6초 달성]
- 문제: SSR 시 느린 API(최대 3초) 하나가 페이지 전체 응답을 블로킹
- 해결: 기본적으로 메인 콘텐츠는 SSR, 서브 콘텐츠는 CSR로 분리하여 FCP를 확보. API 응답이 느린 페이지는 전체 CSR로 전환하여 로딩 UI를 먼저 노출한 뒤 호출
- 성과: 페이지 전체 블로킹을 제거하여 FCP 0.6초 달성, API 응답 속도 기준의 렌더링 전략을 수립하여 신규 페이지 설계 시 판단 기준으로 활용

[계층형 카테고리 필터 시스템 구현]
- 문제: depth를 알 수 없는 계층형 카테고리 필터 구현 (예: 의류 체크 시 아우터·드레스 등 모든 하위 카테고리까지 체크)
- 해결: 부모 체크 시 모든 자식 체크, 자식이 모두 체크되면 부모 자동 체크되는 로직을 재귀로 처리. 제출 시에는 자식을 제거하고 부모 카테고리만 추출하여 전송
- 성과: 5depth 계층 구조를 지원하는 필터 시스템 구현

[프론트 팀 리빌딩]
- 문제: 프로젝트 시작 후 프론트 팀원 전원 퇴사
- 해결: 팀의 문제점을 회사에 투명하게 공유하고, 페이지별 공수를 재산정하여 필요한 인력 규모를 경영진과 협의. 신규 입사자가 빠르게 실무에 투입될 수 있도록 프로젝트 히스토리와 레거시 코드를 문서화하고 멘토링을 주도
- 성과: 일정 내 프로젝트 완수
