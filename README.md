## 제 소개

개발자로서 세 가지를 추구합니다.

1. **사용자 가치** — 사용자는 가치에 돈을 지불합니다. 그래서 저는 사용자에게 가치를 제공하고 싶습니다.
2. **성능과 버그 없는 프로그램** — 빠르고 안정적인 서비스가 좋은 제품의 기반이라고 생각합니다.
3. **생산성** — 모노레포, AI, 베스트 프랙티스를 활용하여 생산성을 올리고 싶습니다.

> [AI 시대에 저는 이렇게 성장하고 있습니다](docs/vision/ai-era-growth.md)
